# フロントエンド実技試験 - 60分

このプロジェクトは、フロントエンドエンジニアのスキルを評価するための60分間の実技試験です。

## 📋 試験内容

### 課題 01: ヒーローセクション
- 背景に大きなテキスト（例：「60分」）を配置
- タイトルは縦書きで配置
- 斜めのライン効果またはグラデーション背景
- レスポンシブ対応（PC・タブレット・モバイル）

### 課題 02: ニュースリスト
- 3枚以上のカード要素を並べる
- 各カードに画像、タイトル、説明を配置
- ホバー時にスケール・シャドウ・テキスト色が変化するアニメーション
- プレースホルダー画像を使用

### 課題 03: ベネフィットセクション
- 3カラムレイアウト（数字、タイトル、説明）
- 大きな数字にグラデーション色を適用
- Intersection Observer を使用したスクロール連動アニメーション
- 数字の背景に装飾（円形、背景画像など）を追加

## 🎯 評価基準

1. **コード品質** - BEM命名規則、適切なHTML構造
2. **ビジュアル** - 指定されたデザイン要素の実装度
3. **レスポンシブ** - モバイル・タブレット・PCでの表示
4. **アニメーション** - スムーズで自然な動作
5. **パフォーマンス** - 不要なリフロー・リペイント回避

## 📝 使用技術

- HTML
- CSS (CSS変数、Flexbox/Grid推奨)
- JavaScript (Vanilla JS のみ)

## ⚠️ 制限事項

- 外部ライブラリ（Bootstrap、jQuery等）の使用禁止
- 素のHTML / CSS / JavaScriptのみ使用可能

## 🚀 使用方法

1. このサイトにアクセス
2. 「試験を開始する」ボタンをクリック
3. HTML / CSS / JavaScriptエリアに実装を入力
4. リアルタイムプレビューで動作確認
5. 「提出コードをコピー」で回答をコピー

## ⏱️ 試験時間

- **60分** - 開始時刻が記録され、ブラウザを閉じても時間が継続します
